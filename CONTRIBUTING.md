# Contributing

Thanks for helping shape Auralith369.

## Development
1. Install dependencies: `npm install`
2. Start dev server: `npm run dev`
3. Build before PR: `npm run build`

## Guidelines
- Preserve local-first behavior.
- Keep user-facing branding as **Auralith369**.
- Prefer minimal dependencies.
- Document `.auralith` format changes in `docs/FILE_FORMAT.md`.
