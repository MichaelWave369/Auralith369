# Auralith369 v0.1.0-alpha

Auralith369 is now live as a public alpha.

**Auralith369** is a local-first visual alchemy workstation by PHI369 Labs for image editing, poster forging, style cards, manifests, and auditable creative receipts.

Online demo:

https://michaelwave369.github.io/Auralith369/

## Highlights

- Public MIT-licensed alpha release
- Live GitHub Pages demo
- Vite + React application shell
- Canvas-based editor workspace
- Layer panel and tool rail foundation
- Hardened `.auralith` project validation
- `.auralith` project save/load foundation
- `.auralith-receipt.json` creative receipt export
- `.auralith-manifest.md` manifest export
- PHI / 369 / spiral overlay foundation
- Style-card and poster/social export foundations
- Node-native validation tests
- GitHub Actions CI
- GitHub Pages deploy workflow
- Pre-React boot diagnostics and React ErrorBoundary
- Public alpha layout polish

## Alpha warning

This is early alpha software. It runs locally in the browser, but the project/file format and UI are still evolving.

Avoid opening untrusted `.auralith` files until validation has matured further.

## Known limitations

- Some Canvas operations are still main-thread and may slow down on large files.
- The editor still contains prototype inline styling in places.
- True non-destructive adjustment layers are planned but not complete.
- Plugin parameters and local AI/ComfyUI bridge are future roadmap items.
