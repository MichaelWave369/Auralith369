# Auralith369 Alpha Notes

Auralith369 v0.1.0-alpha is an early public alpha of a local-first visual alchemy workstation.

## What works now

- Local browser-based editor shell
- Canvas workspace
- Layers panel
- Tool rail
- Project save/load foundation
- Hardened .auralith project validation
- Creative receipt export
- Manifest export
- PHI / 369 / spiral overlay concepts
- Style-card and poster-forge foundations
- GitHub Pages online demo
- CI build/test verification

## Alpha cautions

- This is not production-stable software.
- Avoid opening untrusted .auralith files until validation has matured further.
- Some image operations still use main-thread Canvas APIs and may be slow on large images.
- The editor still contains some inline prototype styling that will be refactored over time.
- Public API and file formats may change before stable release.

## Planned next improvements

- Better layout/design tokens
- Stronger .auralith schema versioning
- Web Worker pixel engine
- True non-destructive adjustment stack
- Parameterized plugin registry
- Local AI / ComfyUI bridge
- Desktop packaging exploration
